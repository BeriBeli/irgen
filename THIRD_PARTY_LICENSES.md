This repository includes Accellera IEEE 1685-2022 IP-XACT XML Schema
Definition files for code generation and XSD validation.

## Accellera IP-XACT Schemas

Paths:

- `crates/ipxact/schema/1685-2022/`
- `crates/ipxact/schema/LICENSE.accellera`
- `crates/ipxact/schema/NOTICE.accellera`

Sources:

- <https://www.accellera.org/XMLSchema/IPXACT/1685-2022/>
- <https://www.accellera.org/XMLSchema/IPXACT/LICENSE>
- <https://www.accellera.org/XMLSchema/IPXACT/NOTICE>

The 2022 XSD files carry Apache-2.0 license headers. The corresponding
Accellera Apache-2.0 license and notice are included as
`LICENSE.accellera` and `NOTICE.accellera`.

These third-party schema licenses do not define the license for irgen project
source code.
