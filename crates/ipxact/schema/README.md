# IP-XACT Schema Fixtures

This directory contains third-party XML Schema files used by `ipxact-codegen`
and by `ip-xact` integration tests.

## Sources

- `1.4/`: Accellera SPIRIT 1.4 schemas from
  <https://www.accellera.org/XMLSchema/SPIRIT/1.4/>
- `1.5/`: Accellera SPIRIT 1.5 schemas from
  <https://www.accellera.org/XMLSchema/SPIRIT/1.5/>
- `1685-2009/`: Accellera/SPIRIT IEEE 1685-2009 schemas from
  <https://www.accellera.org/XMLSchema/SPIRIT/1685-2009/>
- `1685-2014/`: Accellera IEEE 1685-2014 schemas from
  <https://www.accellera.org/XMLSchema/IPXACT/1685-2014/>
- `1685-2022/`: Accellera IEEE 1685-2022 schemas from
  <https://www.accellera.org/XMLSchema/IPXACT/1685-2022/>

## License Notes

The 1.4, 1.5, 2009, and 2014 XSD files are kept as verbatim copies with their
original license notices in each file header. Do not edit, reformat, subset, or
derive new schemas from those files.

The 2022 XSD files carry Apache-2.0 license headers. `LICENSE.accellera` and
`NOTICE.accellera` are verbatim copies from the Accellera IP-XACT schema
directory and apply to the 2022 schema set.

Project source code licensing is separate from these third-party schema
fixtures.
